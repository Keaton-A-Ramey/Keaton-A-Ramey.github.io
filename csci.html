"RedShout" Writeup

I made this project in the Fall of 2023. This was the culmination of a semester where, in my free time, I took up learning React. After learning the fundamentals through a whole host of mini-projects, I created this forum-style website where users can create log in with Google accounts (thanks to Firebase), create posts, like one another's posts, comment on one another's posts, and delete their own comments.

The most challenging part of this was implementing the comments system. While I was able to find helpful guides to creating everything else, the comments system was something that I had no help for that I made as a way to challenge myself and prove to myself that I was capable and understood how to use React. Beyond that, I had never used cloud services like Firebase up to that point so that was also new for me to figure out. Lastly, I had never used TypeScript either -- I found it really useful, especially in a system like this because JavaScript's usual dynamic typing can lead to a whole lot of annoying issues. In the end, this was a really cool learning experience for me and made me feel not only more confident as a React developer, but also as a developer in general who can pick up new technologies.

Raytracing Project

Completed in Foundations of 3D Graphics, I implemented a variety of features in an interactive Raytracing program. While this class was very difficult, it was incredibly rewarding. It was a kind of problem-solving I had never done before, being much more spatial. Additionally, using matrices was something I had very limited experience with. My only exposure was to them as a concept when learning how gradient descent works in an AI class. In the end, though, I implemented Raytracing and a whole host of other functionalities to create an interactive graphics program that I am pretty proud of.

A detailed list of all implemented functionalities:

  • Distance functions, degree/rad, basic operations (normalization, areaOfTriangle, max, min, approximatelyZero, etc.)
  • Trigonometric/Directional functions (pointOnUnitCircle, directionInRadians)
  • Quadratic Formula/Mapping
  • Vector operations (vector normalization, cosBetween, pointingVector, etc.)
  • Plane/Disk intersections
  • Raytracing
  • Lighting (ambient, diffuse, and specular color calculations, positionalLight illumination)
  • Triangles, cylinders, cones
  • Spotlights (isInSpotlightCone)
  • Textures (UV mapping)
  • Matrix Operations (Addition, multiplication, inversion)
  • 2D Transformations (Translation, rotation, scaling)
  • Pipeline Shading/Hidden surfaces: Hidden surface removal using depth Z-values (processFragment()), lighting
  • Antialiasing
  • Shadows
  • Transparent planes (alpha blending)
